# Magische Gegenstände Identifizieren
| Seltenheit    | Informationen, die der DM gibt                 | Identifizierungsmethode                                  | Beispielbeschreibung                                           |
| ------------- | ---------------------------------------------- | -------------------------------------------------------- | -------------------------------------------------------------- |
| **Common**    | Name und vollständige Effekte direkt offenlegt | Keine weitere Identifizierung nötig                      | „Eine *Potion of Healing*. Heilt 2W4+2 HP.“                    |
| **Uncommon**  | Name und vollständige Effekte direkt offenlegt | Keine weitere Identifizierung nötig                      | „Ein *+1 Langschwert*. +1 auf Angriffs- und Schadenswürfe.“    |
| **Rare**      | Name und grobe Funktion, Details unklar        | *Identify*, Arcana-Check (DC 15) oder kurze Rast         | „Ein *Flammenzungen-Schwert*. Es scheint mit Feuer verbunden.“ |
| **Very Rare** | Vage Hinweise auf Aussehen/Aura, keine Details | *Identify*, Arcana-Check (DC 20) oder Einstimmung        | „Ein Stab mit pulsierender Energie, voller Geheimnisse.“       |
| **Legendary** | Sehr vage Hinweise, nur Aussehen oder Mystik   | *Identify*, Arcana-Check (DC 25), Einstimmung oder Quest | „Ein Amulett mit leuchtenden Runen, strahlt uralte Macht aus.“ |
# Arcana-Checks für Zauberwissen

•  **Beschreibung**: Spieler können einen Intelligenz (Arcana)-Check machen, um Informationen über Zauber zu erhalten, die ein Gegner wirkt (z. B. Name oder Effekt). DC hängt vom Zaubergrad ab (z. B. DC 10 + Zaubergrad).

•  **Beispiel**: Ein Feind wirkt Fireball (3. Grad). Ein Arcana-Check (DC 13) könnte enthüllen: „Das ist ein Feuerzauber, der Flächenschaden verursacht.“

•  Regelbasis: Ergänzt die Regeln für Fertigkeitswürfe (PHB, Seite 177-178).

# Tränke als Bonusaktion

•  **Beschreibung**: Das Trinken eines Heiltranks (z. B. Potion of Healing, DMG, Seite 187) kostet nur eine Bonusaktion statt einer Aktion. Das Verabreichen eines Tranks an einen anderen Charakter bleibt eine Aktion.

•  **Regelbasis**: Ändert die Standardregel für die Verwendung von Gegenständen (PHB, Seite 190).

•  **Hinweis**: Um Balance zu wahren, kannst du dies auf Common-Tränke beschränken, passend zu deiner Hausregel für magische Gegenstände.

# Flexible Metamagic für Sorcerer

•  **Beschreibung**: Ein Sorcerer kann einmal pro langer Rast eine gewählte Metamagic-Option gegen eine andere austauschen (z. B. Quickened Spell gegen Twinned Spell). Dies muss während der Rast entschieden werden.

•  **Regelbasis**: Modifiziert die Metamagic-Regel (PHB, Seite 101), die normalerweise festgelegte Optionen vorsieht.

•  **Beispiel**: Der Sorcerer tauscht Empowered Spell gegen Subtle Spell, um in einer sozialen Begegnung unauffälliger zu zaubern.
